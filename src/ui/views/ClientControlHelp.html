Client management: Help

General description

This window allows you to perform basic maintenance operations of the user data. The user may, by means of the controls controls available in the window to create, modify and delete the users of the application. the application. The user can also generate a report containing a list of the users with their corresponding data. of the users with their corresponding data.

Operations

Create clients

To create a user, click on the icon at the top left of the table, this will create a new user with default values.

Modify client data

To modify the data of a client you must double click on one of the cells, this way the edit mode will be enabled, with the edit mode enabled you only have to write the value you want, once you finish modifying the cell, in case you want to save the changes just press the enter key, on the contrary in case you want to cancel it press the escape key.

Delete clients

To delete a client you only have to right click on the user you want to delete, this will open a contextual menu with only one option, which is to delete, once clicked you will be asked for confirmation to delete the selected client, if you cancel no change will be made, on the contrary if you accept, the user will be deleted.

Filters

You have some options for filtering customers, first you can type in the search bar the username, email or full name of the customer to search, this will show in the table all customers whose data match the specified search value, you can also use a filter, on the right side of the search button you have a filter button with two options, enabled and disabled, clicking on any of these will show in the table all customers with the specified filter.

Print list of clients

Pressing the Print button, located at the top right corner of the table, will open a window where you can view a document containing a list of the users that exist in the application:
Using the buttons at the top of the window, you can print or save the report.

Menu

You can navigate between the different administrator windows using the menu on the left side of the window, simply click on the window you want to go to.

You can also Log out simply by clicking the "Log Out" button at the left bottom corner of the window

Close

Pressing the button "X" at the top right corner you can close the app, by clicking it will open a window to confirm wether you are sure if you want to close the app or not, if you accept the app will close and if you cancel the app will continue working normally

Help

Pressing the button Help, located next to the windows title, you will see this help document.
